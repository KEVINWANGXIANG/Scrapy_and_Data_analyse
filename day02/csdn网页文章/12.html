【观察】VMware：二十而冠，以梦为马不负韶华

申耀的科技观察

读懂科技，赢取未来！


今年7月，VMware首席执行官Pat Gelsinger成功登顶了世界七大高峰之一的乞力马扎罗山，期间的巨大挑战可想而知。但越困难越危险，也就越迷人，因为登顶的魅力，也是挑战自我、超越自我的过程。


这似乎也正是VMware这家成立于1998年的美国硅谷公司，在过去二十年成长过程中的一种真实“写照”，从服务器虚拟化，到软件定义数据中心，再到“云中管家”，VMware始终站在市场转型的前沿，并不断实现了自我的迭代与进化。



正如Pat Gelsinger在VMworld2018上所言：“在这个颠覆性变革的时代，VMware致力于帮助客户和合作伙伴拥抱数字化转型，在未来我们仍将继续帮助企业攻克多云管理的复杂性，成为客户与合作伙伴通向数字化转型过程中的桥梁。”


毫无疑问，VMware是与生俱来的市场“颠覆者”，如今二十而冠的它，也正“以梦为马不负韶华，继往开来再次出发。”


云中管家，挑战极限超越自我


时间回到2016年，此时对VMware而言无疑是一个艰难的时刻，当时戴尔正式宣布收购了VMware母公司EMC，而VMware也面临着全面云化转型的挑战，对于过去“战无不胜”的VMware而言，未来怎么走无疑是外界关注的话题。


但经过三年的不断努力，在Pat Gelsinger的带领下，VMware公司不仅成功实现了转型，而且其“云中管家”的梦想也正在加速落地，我们可以从三个维度来做观察：



一是，在混合云领域，VMware正以vSAN为基础，以超融合为平台，帮助企业客户将其本地部署的数据中心快速无缝地拓展至云端，为企业客户实现私有云向混合云，乃至公有云的平滑过渡提供了有力的支撑。


目前，VMware提供了业内最广泛的超融合解决方案，其中既有一站式的超融合集成系统，如Dell EMC VxRail和VxRack SDDC 解决方案、也有经过认证的 vSAN ReadyNode硬件参考架构，更有通过AWS和IBM提供的基于vSAN的超融合云服务。换句话说，VMware vSAN在超融合市场具备“全栈”式的交付能力。


二是，在公有云领域，VMware通过与亚马逊AWS的深入合作，提供VMware Cloud on AWS 服务，支持企业客户公有云环境下运行各种VMware应用。


在今年的VMworld2018上，亚马逊AWS首席执行官Andy Jassy也再次与Pat Gelsinger共同亮相，无疑也体现了双方合作的深度和广度都在不断延展。



可以看到，VMware Cloud on AWS已在美国、欧洲以及包括亚太在内的全球多个地区推出。其中，最新上线的亚太（悉尼）地区还提供了全新功能，包括Elastic DRS、Log Intelligence Service和Cost Insight migration assessment等全新服务。


与此同时，双方还宣布了Amazon RDS可以运行在VMware上，它可以将运行于VMware软件定义数据中心环境或混合云中的数据库迁移到AWS或VMware Cloud on AWS中，让数据库的创建、操作和扩张变得更加容易。


除此之外，通过VMware Cloud on AWS Solution Competency项目，VMware也正在帮助合作伙伴向客户提供公有云服务，目前全球已有近150个合作伙伴加入该项目。同时，针对VMware Cloud on AWS，经测试和验证的相关解决方案更提高了4倍，增至近100个。



三是，在边缘计算领域，VMware也正以VMware Edge为基础，把自身的应用覆盖到了IOT领域，实现了从虚拟化到云计算，从IT到OT的不断拓展。


总的来看，如今的VMware把私有云、混合云、公有云融三为一，帮助用户实现了“跨云”环境管理的挑战，同时还把这种能力延展到了未来的边缘计算领域，让“云中管家”的称号变得真正“名副其实”。


创新不止，投资未来拥抱未来


在Pat Gelsinger看来，云计算、移动、人工智能、物联网将是未来四大“技术超能力”，它将彻底的改变传统的IT，让技术的发展拥有了更多的想象力。所以，VMware也将继续坚持“任意云、任意应用和任意设备”的战略，通过不断的技术创新，投资未来拥抱未来，具体而言：


第一，在数据中心基础设施方面，VMware预览了Project Dimension项目，这是一项涵盖VMware Cloud、数据中心、分支机构和边缘的VMware的全新技术。



据介绍，Project Dimension将以超融合的形式把VMware Cloud Foundation与VMware Cloud托管服务相结合，把VMware运营的SDDC架构作为端到端服务进行交付。未来，Project Dimension可以让数据中心基础设施的交付变得更加简单和高效。


第二，在未来网络的构建方面，今年VMware正式对外发布了“虚拟云网络”（Virtual Cloud Network）战略，首次清晰地描绘了未来网络的新愿景。


全新推出的VMware NSX 网络与安全产品组合，为软件定义的数据中心、各分支机构、边缘网关、云以及电信环境中的应用程序和数据提供无处不在的持续连接性和安全性。它让VMware构筑了一个具有端到端服务能力的网络新平台，也让VMware完成了从网络虚拟化到虚拟云网络的价值新跃迁。



第三，在人工智能方面，全新的Project Magna项目将基于机器学习的自动驾驶数据中心变为可能。


在VMworld2018上，VMware演示说明了Project Magna如何学习和理解应用程序行为，从而能对网络进行建模、测试和重新配置，使其更优化以提升性能。此外，Project Magna依靠人工智能算法，帮助在庞大的数据集之间建立联系，获得从应用程序代码到软件，再到硬件架构，直至公有云和边缘的各种应用和堆栈程序的深入洞察，这将让未来的数据中心变得更加智能化和自动化。


第四，在区块链方面，VMware还演示了一个用于数字共识和智能合约执行的高度可扩展、节能的分散式信任架构Project Concord，它可支持包括区块链在内的分布式信任架构。同时，该项目还将以开源的形式对外提供，为客户打造基于企业级的区块链服务打下了基础。



不仅如此，在容器、安全、数字化工作空间等等领域，VMware也有全新的项目和技术落地和推进。同时，VMware每年特设的VMware Hands-on Labs，其规模也越来越大，如今借助VMware全球公有云资源，Hands-on Labs给开发者也带来了最新的VMware技术体验。


从这个角度来说，VMware的梦想、格局和视野更加长远，它不止于眼下，更在于未来。


二十而冠，以梦为马不负韶华


回头来看，从1998年到2018年，过去二十年的时间，VMware从服务器虚拟化、到软件定义数据中心、再到全面云化时代的“云中管家”，可以说是一个不断颠覆自我，超越自我的历程，这背后无疑也印证了VMware在技术创新力和前瞻洞察力，以及作为市场领导者和颠覆者所具备的强劲实力。



一是，在服务器虚拟化时代，作为服务器虚拟化市场的领袖，VMware成立至今帮助了千千万万的企业客户实现了数据中心基础设施的转型，而它自身也凭借虚拟化技术称雄业内，并让服务器虚拟化成为各行各业数据中心用户的“标配”管理工具。


二是，在软件定义数据中心时代，VMware将其领导地位向真正的软件定义数据中心不断延伸。其中，通过vSAN带来的超融合解决方案，VMware催生了市场中软件定义存储和分布式存储的浪潮，并为传统企业级存储市场带来颠覆性变革；而通过NSX，VMware加速了网络的价值向虚拟化和软件定义的方向转移，打通了软件定义数据中心旅程中的“最后一公里”，也真正为构建企业私有云和下一代数据中心打下了坚实的基础。


三是，在全面云化时代，面对云计算的大势索取，以及多云和跨云环境的下一代挑战，VMware再战江湖，并凭借云端技术的再创新、在定义，把虚拟化技术成功带上云端，成为了全球市场中的“云中管家”。



在此过程中，VMware的业绩也得以不断实现增长，而这也正是市场颁发给VMware的“军功章”。回头来看，VMware刚刚上市时年收入不到4亿美金，但VMware最新的2018年财年财报显示，其年总收入高达79.2亿美元，这种成长速度无疑是一种骄傲，也是一种传奇。


全文总结，如今VMware正以全新的思维方式和超强的技术能力思考云、发展云，并实现了云中的加速奔跑，这是VMware的独特之处，更是其可怕之处。所以，二十而冠的VMware，属于这个伟大的时代，可谓正青春正向上！
